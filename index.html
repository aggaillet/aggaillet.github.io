Eryantis digital

Eryantis digital is the transposition as videogame of the boardgame Eryantis by Cranio Creations.

It was developed by two friends and I as a semester project at Politecnico di Milano.

The game can be self-hosted and played by up to 3 people over an internet or LAN connection by means of a graphical interface or via command line.


Navigation and Attitude System

The Navigation and Attitude System of the Pyxis rocket is responsible for real-time calculation of the rocket's attitude using data from the on-board sensors.

This data is then used to perform altitude control and is trasmitted to the ground station via an array of antennas to allow for flight monitoring.

The estimation is done using a Multiplicative Extended Kalman Filter containing a lightweight rocket model.

The same system is algorithm is used on the guided payload to feed information to the autonomous guidance system.

You can see a (now old-ish) version of the system in action here, and a full flight with the resulting telemetry here.




Sustainable taxiing

Project focusing on the conceptualization and requirement definition for a sustainable taxiing system based on hybrid towing vehicles including, but not limited to, the drafting of a ConOps document and a User Requirement Document.

A safety analyisis was also performed using both System Theoretic Process Analyisis and Functional Hazard Assessment methodologies.



Can a PC diagnose diabetes?

A project presented to the Canada Wide Science Fair in 2019, thanks to a delegation award sponsored by Doctors Manitoba, after winning the prize as Best biological science individual project award sponsored by Bayer Canada and the Canadian Medical Biological Engineering Society Certification of Appreciation award at the provincial level.

This project focused on applying artificial intelligence algorithms to link factors to disease and create a model useful for an initial and cost effective population-wide screening for diabetes.

In particular it used data from the National Health And Nutrition Examination Survey by the CDC to create a 10-factor predictive model.
